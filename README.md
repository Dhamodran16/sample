<title>System Overview</title>
<para>The XJ-900 Turbofan is a high-bypass engine designed for long-range commercial aircraft. It consists of a single-stage fan, a four-stage low-pressure compressor, and a ten-stage high-pressure compressor.</para>
<para>Proper maintenance of the airflow path is critical to engine efficiency. Technicians must ensure that no Foreign Object Debris (FOD) is present in the inlet cowl prior to operation.</para>
</levelledPara>
 
			<levelledPara>
<title>Required Equipment</title>
<para>Before commencing any maintenance task, ensure the following tooling is available at the workstation:</para>
<randomList listItemPrefix="pf02">
<listItem>
<para>Torque Wrench (Calibrated to ISO standards).</para>
</listItem>
<listItem>
<para>Borescope Kit Type-4.</para>
</listItem>
<listItem>
<para>Safety wire pliers.</para>
</listItem>
<listItem>
<para>Protective gloves and eye wear.</para>
</listItem>
</randomList>
</levelledPara>
 
			<levelledPara>
<title>Safety Warnings</title>
<para>The ignition system produces high voltage. Ensure the system is grounded before touching the igniter plugs.</para>
<para>
<emphasis>DANGER:</emphasis> Do not stand within the ingestion zone while the engine is running. The suction force is sufficient to pull in a human being.
</para>
</levelledPara>
